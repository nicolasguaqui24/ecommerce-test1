# Ecommerce en desarrollo
  Se consume API externa, para productos (no hardcodeados)
  Buscar productos en tiempo real
  Pagina resultado de productos, en 6 articulos por card
  Detalla producto ->vista previa o agrega al carrito
  Persistencia del carrito por mas que se recargue la pagina()
  Responsive

# Tecnologias: 
Para la produccion, se utilizaron 
React
TypesScript
Vite (Herramienta de desarrollo)
CSS (por componentes)
Fake Store API (https://fakestoreapi.com) (consumo api externa, para productos:  recibo en formato JSON)

# Requisitos 
node.js 
 -(Descargar desde la web) -https://nodejs.org 


# Instalación
Para instalacion(Pasos a seguir)
1.Clonar repositorio
2.Instalar dependencias (npm install)
3.Correr en modo desarrollo (npm run dev)
4.Abrir navegador, segun indique localhost (url), al correr npm run dev.
